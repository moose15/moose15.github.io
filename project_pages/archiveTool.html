Wiki Archive Tool
Role: Designer, Developer
Date Developed: Summer 2016
Language: JavaScript, HTML5, CSS3
During my second internship at DreamWorks Animation SKG, I helped create an interactive visualization for archiving outdated wiki pages. This tool uses the D3.js JavaScript library to turn data via a JSON file into a diagram of wiki pages. Each diagram showcases a page's title, age, and place in the hierarchy of the selected wiki space. Based on this information, employees may create a list of pages they wish to archive. The tool offers a number of other features as well such as switching the style of the diagram and changing the background color of the canvas.
The diagrams shown in these screenshots show data from a mock-JSON file with dog breeds for page titles.
